# Java Composition Generator

Java program demonstrating object-oriented programming using multiple classes to generate structured text compositions (Composition, Paragraph, and Sentence).

## Features
- Generates compositions consisting of paragraphs and sentences
- Demonstrates interaction between multiple classes
- Uses ArrayLists to manage collections of paragraphs and sentences

## Concepts Used
- Object-Oriented Programming (OOP)
- Classes and objects
- ArrayLists
- Program structure across multiple files

## Class Structure

Driver.java – Runs the program

Composition.java – Manages the list of paragraphs and controls composition creation

Paragraph.java – Handles the creation and storage of sentences within a paragraph

Sentence.java – Represents individual sentences in the composition

## How to Run

1. Compile the files:

javac *.java

2. Run the program:

java Driver
